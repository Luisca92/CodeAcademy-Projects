HTML Tags Cheatsheet

Tag Name Description
<html> HTML Tag Represents the root of an HTML document.
<head> Head Tag A container for metadata (data about data) and is placed between the <html> tag and the <body> tag.
<title> Title Tag Defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.
<body> Body Tag Defines the document's body. The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
<footer> Footer Tag Defines a footer for a document or section.
<h1> Section Heading #1 A heading that's most commonly used to mark up a web page title.
<h2> Section Heading #2 Defines the second level heading in the HTML document.
<h3> Section Heading #3 Defines the third level heading in the HTML document.
<h4> Section Heading #4 Defines the fourth level heading in the HTML document.
<h5> Section Heading #5 Defines the fifth level heading in the HTML document.
<h6> Section Heading #6 Defines the sixth level heading in the HTML document.
<p> Paragraph Tag Defines a paragraph. Browsers automatically add a single blank line before and after each <p> element.
<br> Line Break Tag Inserts a single line break.
<hr> Horizontal Rule Tag Defines a thematic break (e.g. a shift of topic). The <hr> element is most often displayed as a horizontal rule that is used to separate content (or define a change).
<section> Section Tag Defines a section in a document. A semantic element for creating standalone sections in a web page. These sections should be made up of related content, like contact information. The section element should only be used if there isn't a more specific element to represent the related content.
<div> Division Tag defines a division or a section in an HTML document. The <div> tag is used as a container for HTML elements. The <section> tag means that the content inside relates to a single theme, and <div> tag is used as a block part of the webpage and don't convey any particular meaning.
<span> Span Tag An inline container used to mark up a part of a text, or a part of a document.
<a> Anchor Tag The most important attribute of the <a> element is the href attribute, which indicates the link's destination.
<form> Form Tag The <form> tag is used to create an HTML form for user input.
